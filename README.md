# timeular-cli

A timeular cli written in rust

# Features

- [X] Printing the current activity
- [ ] Starting an activity
- [ ] Stopping an activity
- [ ] Adding an activity
- [ ] Everything else
- [ ] Some kind of TUI

# Installation

```sh
$ git clone https://github.com/ragon000/timeular-cli
$ cd timeular-cli
$ cargo build --release
$ sudo cp target/release/timeular-cli /usr/local/bin/
```
